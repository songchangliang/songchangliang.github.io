3.EJS基础语法：

​ 看到这里可能有很多人会说，你不是说基本没有特殊语法吗？ 不是的，基本没有不是说就没有，只是很简单，一说就会：

  • ejs.render(注入文件, 注入信息) :

    ​ 此方法是EJS的核心语法，用来将数据注入到文件

  • <% %> :

    ​ EJS的模板标签，写EJS时必须要将html文件都后缀改为.ejs，而这个模板就是用来接传来的数据的，

    ​ 也可使用 <? ?> 来代替 <% %>

  • <%= %> :

    ​ 用来输出数据到模板

  • <%# %>:

    ​ 注释标签，不执行、不输出内容
